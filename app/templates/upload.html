Petunjuk Format Data

Untuk memastikan pipeline preprocessing berjalan tanpa hambatan, file Excel Anda sebaiknya memenuhi ketentuan berikut:

  • Setiap Sheet merepresentasikan satu wilayah Desa (nama sheet akan otomatis digunakan sebagai nama desa).
  • Format baris header di setiap sheet dapat bertingkat (multi-header), asalkan kolom-kolom utama seperti Nama, Umur, Jenis Kelamin, BB, TB/IMT, TD, Gula Darah, Kolesterol teridentifikasi.
  • Kolom umur dapat dikelompokkan (misal kolom "45-59 L", "45-59 P"), sistem akan mengekstraksi dan menyatukannya secara otomatis.
  • Data yang tidak valid (misal baris TOTAL atau baris kosong) akan dibuang secara otomatis selama proses parsing.
